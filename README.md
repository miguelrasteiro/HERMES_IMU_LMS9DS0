# IMU_LMS9DS0
 - A driver for LSM9DS0 STMicroeletronics 9 DoF axis sensor. 
 - Tested with a PIC32MX795F512L.
 - I2C communication.
 - C Language.
 
## Usage
 For general use. Only has to be adapted to specific microcontroller.
